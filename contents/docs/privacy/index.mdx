## Privacy on Solana

This section covers privacy-focused tools and techniques for building on Solana.

### Overview

Solana offers several privacy-enhancing technologies:

- **Confidential Transfers** - Token transfers with encrypted amounts using the Token-2022 program
- **Zero-Knowledge Proofs** - Cryptographic proofs that verify information without revealing it
- **Light Protocol** - Compressed accounts with privacy features
