## Deposit SOL

Deposit SOL into the Privacy Cash privacy pool. Your deposit generates a commitment that is added to a Merkle tree, enabling private withdrawals later.

### Example with @solana/kit

```typescript
import { deposit } from 'privacycash';
import { createSolanaRpc, createSolanaRpcSubscriptions, createKeyPairSignerFromBytes, lamports } from '@solana/kit';
import wallet from './wallet.json';

const rpc = createSolanaRpc('https://api.mainnet-beta.solana.com');
const rpcSubscriptions = createSolanaRpcSubscriptions('wss://api.mainnet-beta.solana.com');

const depositSol = async () => {
    const keypairSigner = await createKeyPairSignerFromBytes(new Uint8Array(wallet));

    // Deposit SOL into the privacy pool
    const result = await deposit({
        rpc,
        payer: keypairSigner,
        amount: lamports(1_000_000_000n), // 1 SOL in lamports
    });

    console.log('Deposit successful:', result);
    console.log('Save your note securely:', result.note);
};

depositSol();
```

### How it Works

1. You deposit SOL into a shielded pool
2. A cryptographic commitment is generated
3. The commitment is added to a Merkle tree on-chain
4. You receive a note/secret that proves ownership
5. This note allows you to withdraw to any address without linking transactions

### Important

- Save your deposit note securely - it's required for withdrawal
- Deposits are irreversible once confirmed
