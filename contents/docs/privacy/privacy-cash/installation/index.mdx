## Install Privacy Cash SDK

Privacy Cash SDK enables developers to deposit, withdraw, and query balances in the Privacy Cash Solana program.

### Requirements

- Node.js version 24 or above

### Install via npm

```bash
npm install privacycash @solana/kit
```

### Basic Setup with @solana/kit

```typescript
import { deposit, withdraw, getPrivateBalance } from 'privacycash';
import { createSolanaRpc, createKeyPairSignerFromBytes } from '@solana/kit';
import wallet from './wallet.json';

// Setup RPC connection
const rpc = createSolanaRpc('https://api.mainnet-beta.solana.com');

// Load your keypair
const keypairSigner = await createKeyPairSignerFromBytes(new Uint8Array(wallet));
```
