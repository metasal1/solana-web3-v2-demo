## Install Privacy Cash SDK

Privacy Cash SDK enables developers to deposit, withdraw, and query balances in the Privacy Cash Solana program.

### Requirements

- Node.js version 18 or above

### Install via npm

```bash
npm install privacycash @solana/web3.js
```

### Basic Setup

```typescript
import { PrivacyCash } from 'privacycash';

const client = new PrivacyCash({
    RPC_url: 'YOUR_SOLANA_MAINNET_RPC_URL',
    owner: 'YOUR_PRIVATE_KEY'
});
```
