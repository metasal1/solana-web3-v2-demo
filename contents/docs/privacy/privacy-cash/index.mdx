## Privacy Cash

[Privacy Cash](https://privacycash.co/) is a privacy protocol on Solana that enables private SOL and SPL token transactions using zero-knowledge proofs.

### Features

- Private SOL transfers
- Private USDC/USDT transfers
- Zero-knowledge proof based
- Compliance-first design with selective disclosure

### Fees

0.35% + 0.006 SOL per withdrawal
