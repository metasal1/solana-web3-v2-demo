## SPL Token CLI

Command-line tools for managing SPL tokens on Solana.

### Token Management

- [Create Token](/docs/cli/create-token) - Create a new SPL token
- [Create Account](/docs/cli/create-account) - Create a token account
- [Mint Tokens](/docs/cli/mint-tokens) - Mint tokens to an account
- [Burn Tokens](/docs/cli/burn-tokens) - Burn tokens from an account

### Account Operations

- [Close Token Account](/docs/cli/close-token-account) - Close a token account
- [Get Supply](/docs/cli/get-supply) - Get token supply info
- [Display Information](/docs/cli/display-information) - Display token info

### Authority Management

- [Disable Freeze](/docs/cli/disable-freeze) - Disable freeze authority
- [Disable Mint](/docs/cli/disable-mint) - Disable mint authority
